# Icon Button

Icon buttons are compact buttons that contain only an icon, useful for toolbars and action menus.

---

## Installation

{% framework-tabs %}
{% tab framework="react" %}
```bash
npm install @libretexts/davis-react
```
{% /tab %}
{% tab framework="vue" %}
