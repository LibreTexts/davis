## Not Sure Which to Choose?

**Need pre-built components?**
- Building a **React** app? → Choose React
- Building a **Vue** app? → Choose Vue (coming soon)

**Only need design tokens?**
- Want just colors, typography, and spacing? → Choose Tokens Only
- Building custom components from scratch? → Choose Tokens Only
- Using a different framework (Svelte, Angular, etc.)? → Choose Tokens Only

**More guidance:**
- See the [Architecture](/docs/foundation/architecture) guide for detailed package comparisons
- Check out [Existing Projects](/docs/guides/existing-projects) if integrating into an existing codebase

## What You'll Get

All installation paths include:
- **Design tokens** - Colors, typography, spacing, and layout systems
- **Tailwind preset** - Pre-configured design system tokens
- **Base styles** - CSS reset and foundational styles
- **WCAG AA compliance** - Accessibility built-in

Component libraries (React/Vue) additionally include:
- Pre-built, accessible UI components
- TypeScript definitions
- Framework-specific optimizations
- Headless UI integration
